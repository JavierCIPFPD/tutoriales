# Mi Primer Proyecto Web

Proyecto web estatico de una sola pagina que explica cual es la estructura minima recomendable para publicar una web sencilla con un acabado profesional.

## Instalacion

1. Descarga o clona esta carpeta en tu equipo.
2. Abre la carpeta `miprimerproyectoweb` en Visual Studio Code.
3. No necesita dependencias ni proceso de compilacion.

## Uso

1. Abre `index.html` en un navegador para ver la web localmente.
2. Modifica `assets/css/style.css` para cambiar el aspecto visual.
3. Modifica `assets/js/main.js` si quieres ajustar el cambio de tema o el efecto parallax.
4. Sube el contenido de la carpeta a un hosting estatico o a GitHub Pages para publicarlo.

## Estructura

- `index.html`: contenido y estructura de la pagina.
- `assets/css/style.css`: estilos, tema claro/oscuro y diseno responsive.
- `assets/js/main.js`: interacciones basicas de la interfaz.
- `assets/img/favicon.svg`: icono del sitio.
