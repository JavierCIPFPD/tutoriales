# Depuracion con Chrome

Tutorial web estatico de una sola pagina para explicar, paso a paso, como depurar una suma en Chrome DevTools y por que conviene usar el depurador antes que abusar de `console.log`.

## Instalacion

1. Descarga o clona esta carpeta en tu equipo.
2. Abre la carpeta `depuracionconchrome` en Visual Studio Code.
3. No necesita dependencias ni proceso de compilacion.

## Uso

1. Abre `index.html` en un navegador para ver la pagina localmente.
2. Revisa `assets/img` para reutilizar los SVG del tutorial o adaptarlos a otros ejemplos.
3. Modifica `assets/css/style.css` si quieres mantener el mismo estilo visual en otros tutoriales.
4. Usa `assets/js/main.js` para el cambio de tema y el efecto parallax.

## Estructura

- `index.html`: contenido del tutorial y pasos visuales de depuracion.
- `assets/css/style.css`: estilos, layout responsive y componentes de la guia.
- `assets/js/main.js`: interacciones basicas de la interfaz.
- `assets/img/*.svg`: favicon e ilustraciones de los iconos y controles de Chrome DevTools.
