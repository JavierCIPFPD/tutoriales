Ejercicio de depuracion

Suma dos numeros sin distracciones

Abre Chrome DevTools con F12, ve a Sources, pon un breakpoint en la linea de la suma y observa por que `2 + 3` produce `23` en lugar de `5`.

El codigo a depurar

function sumar() {
  const a = document.querySelector('#input-a').value;
  const b = document.querySelector('#input-b').value;
  const resultado = a + b;
  document.querySelector('#output').textContent = resultado;
}
