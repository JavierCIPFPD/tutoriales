# Tutoriales Web

Este repositorio agrupa proyectos web estaticos de una sola pagina, pensados como material didactico para aprender estructura web basica, Git y publicacion en GitHub Pages. En progreso...

## Contenido

- `miprimerproyectoweb`: guia de estructura minima para una web sencilla con acabado profesional.
- `proyectosyrepositorios`: tutorial paso a paso para gestionar un proyecto con Git, conectarlo a GitHub y publicarlo.

## Uso

1. Abre cualquiera de las carpetas de proyecto en Visual Studio Code.
2. Abre su archivo `index.html` en un navegador para ver el contenido local.
3. Edita los archivos dentro de `assets/css` y `assets/js` para personalizar estilos e interacciones.
