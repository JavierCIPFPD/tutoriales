# Proyectos y Repositorios

Proyecto web estatico de una sola pagina que enseña paso a paso como gestionar un proyecto con Git, sincronizarlo con GitHub desde Visual Studio Code y publicarlo con GitHub Pages.

## Instalacion

1. Descarga o clona esta carpeta en tu equipo.
2. Abre la carpeta `proyectosyrepositorios` en Visual Studio Code.
3. No necesita instalar paquetes ni ejecutar comandos de build.

## Uso

1. Abre `index.html` en un navegador para consultar el tutorial.
2. Edita `assets/css/style.css` para ajustar colores, composicion o componentes.
3. Edita `assets/js/main.js` si quieres cambiar el comportamiento del tema o del parallax.
4. Publica la carpeta en un hosting estatico o en GitHub Pages para compartir la guia online.

## Estructura

- `index.html`: contenido del tutorial, infografia y pasos guiados.
- `assets/css/style.css`: estilos comunes del sitio y componentes de la guia.
- `assets/js/main.js`: cambio de tema y efecto de desplazamiento.
- `assets/img/favicon.svg`: icono del proyecto.
